# Sankofa Alkebulan

**Advancing African Scholarship, Policy, and Societal Transformation**

---

## Overview

Sankofa Alkebulan is an institutional platform dedicated to advancing African scholarship, influencing policy, and promoting societal transformation across the continent. This repository contains the official website of the organization, designed to showcase our research, programs, events, and community initiatives.  

The website is fully responsive, accessible, and built using **HTML, CSS, and JavaScript**, making it easy to maintain and expand. It is hosted via **GitHub Pages** and can be linked to a custom domain if desired.
